An HTML Cheat Sheet for Creating Tables

HTML Tag Description
< table > Creates the table
< thead > Creates the table's header
< tbody > Creates the table's body
< tfoot > Creates the table's footer
< tr > Creates a row in the table
< th > Populates the table header with data
< td > Populates the table body with data
Attributes Used in Tables
colspan Defines how many columns a cell should span
rowspan Defines how many rows a cell should span
